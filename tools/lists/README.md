# ./lists

This directory is designed to hold lists of AI Slop sources, and it holds utilities for retrieving online lists. It may also eventually contain tools for ETL operations on lists.

The lists in this directory are the "canonical" versions of the lists as of 2026-03. Running ./build.sh updates the "hardcoded" lists in /extension.

Eventually lists will be pulled from the web, but we'll cross that bridge when we get to it.
